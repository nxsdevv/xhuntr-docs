# Contributing to XHuntr Docs

Thanks for your interest in improving XHuntr's documentation.

## What this repo is

This repository contains documentation, guides, and resources for [XHuntr](https://xhuntr.com) — a Telegram bot that monitors X (Twitter) community activity for Solana alpha traders.

## How to contribute

### Fixing typos or errors
Open a PR directly. Small fixes are always welcome.

### Adding guides or use cases
If you've developed a workflow with XHuntr that others would benefit from, open an issue first to discuss before writing a full guide.

### Reporting outdated information
Open an issue with the label `outdated`. XHuntr is actively developed — features, pricing, and bot commands change. Reports help keep this accurate.

## Pull Request guidelines

1. One topic per PR — don't bundle unrelated changes
2. Use clear commit messages: `docs: fix typo in setup guide`, `docs: add convergence alert example`
3. Check that any links you add are working

## Issues

Use GitHub issues for:
- Outdated documentation
- Missing documentation for a feature
- Broken links or screenshots

For **bot support**, questions, or bugs with XHuntr itself, use Telegram: [@lukefrostdev](https://t.me/lukefrostdev)

## Code of conduct

Be respectful. This is a small project.
